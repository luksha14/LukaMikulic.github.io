Here's some of my Projects

Catan Simulator

A Python application that recreates the Catan board game with advanced map generation, rule validation and game simulations. It features automatic balanced map generation (resources, numbers and harbors) following official Catan rules and includes simulations to evaluate the best settlement spots and expected yields. The project also supports a bot player that can autonomously play the game to a target victory point score. It was built with Tkinter for the user interface and Matplotlib for board visualization.

CannonBall Simulation

Built in the Haxe programming language using OpenFL, this interactive 2D simulation models real-world physics including gravity, air resistance (based on projectile diameter), wind, mass, velocity and launch angle/height. It features trajectory comparison with and without drag, as well as a target practice mode where the user inputs target coordinates and attempts to hit it.

Somnia 360° Photobooth

A responsive website built for Somnia 360° Photobooth, showcasing services, image and video galleries and an integrated contact form with direct email functionality. The project is primarily front-end but also includes a simple back-end feature for handling messages.

RIdeVoyage

An academic project developed in C#, designed to simulate a car rental system. The application allows users to add and remove cars, select rental dates, make reservations and leave reviews. It demonstrates key concepts of software development, including data handling, user interaction and basic CRUD functionality.

Crowdfunding Startup - Django App

A Django web application with user authentication, role-based authorization (admin and regular user) and full CRUD functionality. The system includes user management, data browsing with filtering and search, as well as the ability to add, edit and delete objects. Additionally, I implemented a RESTful API using Django REST Framework, providing secure endpoints for basic CRUD operations with authentication.
